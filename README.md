## 📝 Customization Guide

### Add More Millets
Edit `js/app.js`:
```javascript
const products = [
    { id: 1, name: "Organic Foxtail Millet (Thinai) 500g", price: 89, category: "millets", ... }
];
```

### Change GST Rate
Edit `js/payment.js`:
```javascript
const gstRate = 0.05; // 5% for food items
```

### Add Delivery Zones
Edit `js/payment.js`:
```javascript
const deliveryZones = {
    'your-pincode': { zone: 'Your Zone', express: true, time: '30-45 min', fee: 0 }
};
```
